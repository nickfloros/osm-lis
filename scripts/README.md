
To deploy overpass turbo api do the following 
```
  ./overpass-api.bash  deploy  hampshire

```

to stop turbo api
```
  ./overpass-api.bash stop hampshire
```

to start overpass api
```
  ./overpass-api.bash start hampshire
```
